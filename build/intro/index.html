Python ‘da yeniyseniz, nasıl bir dil olduğu hakkında araştırma yaparak başlayabilirsiniz. Django, 100% Python’dur. Python’da ne kadar bilginiz varsa Django’ya hükmetmeniz o kadar hızlı olacaktır.

Programlamaya yeni başlıyorsanız, programci olmayanlar için Python kaynakları belgelerine göz atarak başlayabilirsiniz.

Diğer birkaç dilde bilginiz varsa Python’u öğrenmeniz hızlı bir şekilde olabilir. Dive Into Python (aynı zamanda kitap versiyonunu’da alabilirsiniz) kitabına göz atmanızı tavisiye ederiz. Bu size göre değilse Python hakkındaki kitaplara bakabilirsiniz.
